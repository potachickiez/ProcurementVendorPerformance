# Vendor Performance Dashboard

## Overview
This project analyzes vendor performance using procurement data.

## Objective
Evaluate vendor performance based on spending, savings, quality, and delivery performance.

## Tools
- Power BI
- Excel
- DAX

## Dataset
Kaggle Dummy Dataset : https://www.kaggle.com/datasets/shahriarkabir/procurement-kpi-analysis-dataset

## Key Insights

- Epsilon Group has the highest compliance rate of 98.2%, while further evaluation is needed for Delta Logistics, which has the lowest compliance rate of 60.8%.
- The vendor's overall score is at a medium level, with a range of 61-73%, with Alpha Inc. as the best vendor.
- Beta Supplies achieved the highest negotiated price reduction and the highest total saving.

## Analysis
### Correlation Analysis
- A Pearson correlation value of 0.42 indicates a moderate positive correlation between negotiated price and quality performance.
- Vendors with higher negotiated prices tend to provide higher quality performance.
## Quadrant Analysis
- Epsilon Group and Gamma Co are premium vendors with prices that tend to be above average and high-quality performance.
- Alpha Inc. is an alternative best vendor that provides high-quality performance at relatively low prices.
- Beta Supplies has relatively high prices with below-average quality performance, requiring further evaluation.
- Delta Logistics is an economical vendor that offers relatively low prices, but also accompanied by low quality performance.
